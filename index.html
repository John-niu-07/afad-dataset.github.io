0. Abstract

The Asian Face Age Dataset (AFAD) is a new dataset proposed for evaluating the performance of age estimation, which is collected from the RenRen Social Network. According to the birth data that RenRen Social Network requires users to provide, labels can be generated.

Compared with other aging databases, e.g. FG-NET which contains about 1k images, MORPH ALBUM II, which contains about 55k images, the AFAD has 160k images. Additionally, there is few ASIAN FACE in the MORPH and FG-NET dataset, thus the AFAD is noticeable in terms of ethic.
